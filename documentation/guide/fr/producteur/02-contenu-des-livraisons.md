# Décrire le contenu d'une livraison

## À quoi ça sert

Préciser, pour une livraison donnée et pour chaque taille de panier, les **items**
inclus et, le cas échéant, leur **poids**. Les amapiens et les coordinateurs peuvent
ainsi savoir ce que contient le panier.

> Prérequis : avoir défini les **items** de votre type de produit (voir
> [Gérer mon catalogue de produits](01-catalogue-produits.md)).

## Ouvrir la description d'une livraison

Depuis votre tableau de bord, ouvrez une livraison et accédez à sa **description**.
Elle est organisée par taille de panier.

```
┌──────────────────────────────────────────────┐
│ ← Légumes Bio — Livraison 17 Jan               │
│  Panier Petit                                  │
│  [img] Carottes        500 g                   │
│  [img] Courgettes      300 g                   │
│        Poireaux        —                       │
│                                          [ + ] │
│  Panier Moyen                                  │
│  [img] Carottes        800 g             [ + ] │
└──────────────────────────────────────────────┘
```

Pour chaque taille de panier, les items présents sont listés avec leur image (si
renseignée) et leur poids (ou « — » si aucun poids n'est précisé).

## Ajouter un item à un panier

1. Sous la taille de panier souhaitée, touchez **[ + ]**.
2. Sélectionnez l'**item** dans la liste (un item déjà présent ne peut pas être ajouté
   deux fois pour la même taille de panier).
3. Renseignez éventuellement le **Poids (g)**.
4. Touchez **[Enregistrer]**.

## Modifier ou retirer un item

- Touchez un item pour **modifier son poids**.
- Faites glisser un item vers la gauche pour le **retirer** de cette taille de panier.

> Cet écran est en **lecture seule** pour les coordinateurs : seuls vous, en tant que
> producteur, pouvez modifier la description.
