# Guide du Producteur

En tant que producteur, vous gérez votre **catalogue de produits** et vous décrivez le
**contenu de vos livraisons** pour les AMAP partenaires.

## Sommaire

1. [Gérer mon catalogue de produits](01-catalogue-produits.md)
2. [Décrire le contenu d'une livraison](02-contenu-des-livraisons.md)

## Votre tableau de bord

Après connexion, votre tableau de bord présente une vue d'ensemble de votre saison
(contrats actifs, volumes), les **livraisons urgentes** à préparer et vos **prochaines
livraisons**. Touchez **[DÉTAILS]** sur une livraison pour en consulter le contenu, et
**[MARQUER PRÊT]** lorsque votre production est prête.
