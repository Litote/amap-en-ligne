# Glossaire

Définitions des termes employés dans l'application et dans ce guide.

| Terme | Définition |
|-------|------------|
| **AMAP** | Association pour le Maintien d'une Agriculture Paysanne : elle crée un lien direct entre des producteurs et des consommateurs autour de produits locaux et de saison. |
| **Amapien** | Membre d'une AMAP. Il peut avoir des contrats avec un ou plusieurs producteurs, s'inscrire comme bénévole et consulter ses contrats. Aussi appelé « bénévole » lorsqu'il participe à une livraison. |
| **Coordinateur** | Membre chargé d'organiser les livraisons : planning, créneaux, contrats de saison, suivi des inscriptions. |
| **Administrateur d'AMAP** | Membre disposant des droits d'administration de son AMAP : gestion des membres, des producteurs et des modèles de livraison. |
| **Administrateur d'instance** | Personne qui pilote le serveur (l'« instance ») : il valide les demandes de création d'AMAP et de comptes producteurs et gère les comptes de l'instance. |
| **Producteur** | Fournisseur de produits (maraîcher, éleveur, boulanger…). Il définit le contenu des paniers. Un producteur peut avoir son propre compte, ou être géré directement par l'AMAP. |
| **Instance / serveur** | Installation de l'application qui héberge une ou plusieurs AMAP. Amap en ligne propose un serveur par défaut, mais chacun peut installer le sien. |
| **Livraison** | Distribution organisée à une date donnée, pendant laquelle les bénévoles préparent les paniers et les amapiens viennent les récupérer. |
| **Créneau de bénévolat** | Plage horaire d'une livraison pendant laquelle des bénévoles sont attendus pour préparer et distribuer les paniers. |
| **Créneau anticipé** | Possibilité, sur certaines livraisons, d'arriver plus tôt que l'heure habituelle pour réceptionner les produits du producteur. Le nombre de places y est limité. |
| **Modèle de livraison** | Paramètres réutilisables d'une livraison (horaires habituels, nombre de bénévoles souhaité, créneau anticipé éventuel), définis par l'administrateur de l'AMAP pour simplifier la création des livraisons. |
| **Panier** | Lot de produits remis à un amapien lors d'une livraison. Une taille de panier (petit, moyen, grand…) correspond à une quantité. |
| **Contrat de saison** | Engagement type proposé par l'AMAP pour un produit donné : taille de panier, période, nombre de livraisons et prix total. Il sert de base pour engager les amapiens. |
| **Contrat d'un amapien** | Engagement individuel d'un amapien sur un contrat de saison. Il précise la période et la taille de panier dont l'amapien bénéficie. |
| **Coordinateur référent** | Coordinateur qui se spécialise sur un contrat de saison (les légumes, le pain…). C'est une information ; elle ne donne aucun droit particulier. |
| **Échange de panier** | Possibilité, pour un amapien absent, de proposer son panier à un autre membre, qui pourra l'accepter. |
| **Feuille d'émargement** | Document imprimable listant les bénévoles à pointer et les paniers à récupérer le jour de la livraison. |
| **Demande d'adhésion** | Demande envoyée par une personne souhaitant rejoindre une AMAP, à valider par un administrateur. |
| **Demande de création** | Demande de création d'une nouvelle AMAP ou d'un compte producteur, à valider par l'administrateur de l'instance. |
| **Invitation** | E-mail envoyé à une personne pour qu'elle active son compte et rejoigne une AMAP (ou devienne administrateur d'instance). |
| **Activation** | Étape où une personne invitée choisit son mot de passe pour finaliser la création de son compte. |
| **Synchronisation** | Échange automatique de données entre votre appareil et le serveur. Elle permet le fonctionnement hors connexion : vos actions sont envoyées dès qu'une connexion est disponible. |
