# Mon historique de participations

## À quoi ça sert

Consulter vos participations comme bénévole : statistiques, prochains engagements et
participations passées.

## Y accéder

Ouvrez le **[Menu]**, puis **[MON HISTORIQUE]** (également accessible depuis l'accueil).

## Ce que vous voyez

```
┌──────────────────────────────────────────────┐
│ 🎯 Mes statistiques                             │
│   Total participations : 8                     │
│   Dernière participation : 3 janvier 2025      │
│                                                │
│ ⭕ Engagements à venir                          │
│   10 Jan 2025 • 18h-20h • ✅ Confirmé          │
│                                                │
│ ✅ Participations réalisées                     │
│   3 Jan 2025 • ✅ Participation confirmée       │
│   13 Déc 2024 • ⚠️ Absence signalée            │
└──────────────────────────────────────────────┘
```

- **Mes statistiques** : nombre total de participations sur la saison et date de votre
  dernière participation.
- **Engagements à venir** : vos prochaines participations confirmées, avec les autres
  bénévoles inscrits.
- **Participations réalisées** : l'historique de vos livraisons passées.

### Les états d'une participation

| Indicateur | Signification |
|------------|---------------|
| ✅ **Confirmé** | Inscription validée, à venir |
| ✅ **Participation confirmée** | Présence effective, passée |
| ⚠️ **Absence signalée** | Absence prévenue, avec remplacement organisé |
