# Guide de l'Amapien

En tant qu'amapien, vous bénéficiez de paniers via vos contrats et vous participez à la
vie de l'AMAP, notamment en vous inscrivant comme bénévole sur les livraisons.

Ce guide couvre toutes vos actions courantes.

## En un coup d'œil

Après connexion, votre **accueil** affiche :

- votre **prochaine participation** comme bénévole ;
- les **prochaines livraisons**, avec la possibilité de vous inscrire en un geste ;
- un résumé de votre **historique**.

Le **[Menu]** (en haut de l'écran) donne accès à l'accueil, à vos contrats, à votre
historique, à vos préférences et à la déconnexion.
