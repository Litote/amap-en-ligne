# Mes livraisons et l'inscription comme bénévole

## À quoi ça sert

Consulter le planning des livraisons de votre AMAP et vous inscrire (ou vous
désinscrire) comme bénévole sur les créneaux de préparation des paniers.

## Depuis l'accueil

Votre accueil met en avant vos prochaines participations et les livraisons qui ont
besoin de bénévoles :

```
┌──────────────────────────────────────────────┐
│ 🎯 Ma prochaine participation                  │
│   Mercredi 31 Jan • 18h-20h                    │
│   ✅ Inscrit(e) • 5/5 bénévoles                │
│   [SE DÉSINSCRIRE]                             │
│                                                │
│ 📋 Prochaines livraisons                       │
│   Mercredi 17 Jan • 18h-20h                    │
│   ⚠️ Besoin urgent • 2/5 bénévoles             │
│   [S'INSCRIRE]                                 │
│                                                │
│ [VOIR PLANNING COMPLET]   [MON HISTORIQUE]     │
└──────────────────────────────────────────────┘
```

- **[S'INSCRIRE]** vous inscrit immédiatement sur le créneau.
- **[SE DÉSINSCRIRE]** annule votre inscription.
- **[VOIR PLANNING COMPLET]** ouvre le planning mensuel détaillé.

## Le planning des livraisons

Le planning affiche les livraisons mois par mois. Utilisez les flèches en haut pour
passer d'un mois à l'autre.

Chaque livraison indique sa date, ses horaires, les producteurs présents, le nombre de
bénévoles et votre éventuelle inscription.

### Les états d'un créneau

| Indicateur | Signification |
|------------|---------------|
| ✅ **Vous êtes inscrit(e)** | Votre participation est confirmée |
| ✅ **TERMINÉ** | Livraison passée à laquelle vous avez participé |
| 🔴 **Besoin urgent de bénévoles** | Moins de la moitié des bénévoles nécessaires |
| ⚠️ **Places limitées** | Le créneau se remplit |
| **COMPLET** | Toutes les places sont prises |
| ❌ **Créneau annulé** | Le créneau a été annulé par la coordination — l'inscription est impossible |

Lorsqu'un créneau est annulé, vos éventuelles inscriptions sur ce créneau sont
automatiquement annulées et vous recevez une notification.

## S'inscrire comme bénévole

1. Ouvrez le **planning** (ou repérez la livraison sur l'accueil).
2. Sur la livraison souhaitée, touchez **[S'INSCRIRE]** (ou
   **[S'INSCRIRE MAINTENANT]** sur un créneau urgent).
3. Votre inscription apparaît immédiatement. Vous pouvez l'annuler à tout moment avec
   **[SE DÉSINSCRIRE]**.

## Les livraisons avec créneau anticipé

Certaines livraisons proposent un **créneau anticipé** : vous pouvez arriver plus tôt
que l'heure habituelle pour réceptionner les produits du producteur. Le nombre de
places y est limité.

Lorsqu'un créneau anticipé est disponible, la livraison affiche **deux boutons**
directement sur sa carte :

```
┌──────────────────────────────────────────────┐
│  Mercredi 31 Janvier • 18h00-20h00             │
│  ⏰ Créneau anticipé disponible                 │
│                                                │
│  [S'inscrire • Créneau standard 18h00-20h00]   │
│                                                │
│  [S'inscrire • Créneau anticipé 17h00-20h00]   │
│  ℹ️ Réception des légumes du maraîcher          │
└──────────────────────────────────────────────┘
```

- **[S'inscrire • Créneau standard]** — l'horaire de bénévolat habituel.
- **[S'inscrire • Créneau anticipé]** — l'arrivée anticipée. L'explication fournie par
  le coordinateur s'affiche juste sous le bouton.

Si le créneau anticipé est complet ou absent, un seul bouton d'inscription est proposé.

## Voir qui coordonne la livraison

Chaque livraison affiche les **coordinateurs** responsables, regroupés par produit
(par exemple 🥕 Légumes, 🍞 Pain). Lorsqu'un coordinateur a indiqué son numéro de
téléphone, vous pouvez le contacter directement en touchant le numéro. Si un produit
n'a pas encore de coordinateur, la mention « Coordinateur à confirmer » s'affiche.

## Panier partagé

Si vous partagez le panier d'un contrat avec une ou plusieurs autres familles (en
alternance), chaque livraison concernée indique si **c'est votre tour** de récupérer le
panier cette semaine (« 🤝 Panier partagé : c'est votre tour de récupérer le panier. »)
ou si c'est une autre famille. Vous retrouvez aussi la liste de vos dates de retrait sur
l'écran [Mes contrats](02-mes-contrats.md). Le partage est mis en place par votre
coordinateur.

## Recevoir des rappels

Vous pouvez choisir d'être prévenu avant un créneau (24 h, 2 h ou 30 min avant) et
recevoir les alertes de besoin urgent. Voir
[Mes notifications et préférences](05-notifications-preferences.md).
