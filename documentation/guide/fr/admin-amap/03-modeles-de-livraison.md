# Gérer les modèles de livraison

## À quoi ça sert

Définir des **modèles** réutilisables qui pré-remplissent les paramètres d'une livraison
(horaires habituels, nombre de bénévoles souhaité, et éventuellement un créneau
anticipé). Les coordinateurs gagnent ainsi du temps lors de la création des livraisons.

## Y accéder

Ouvrez le **[Menu]**, puis **[TEMPLATES DE LIVRAISON]** (ou l'accès rapide
« Templates de livraison »).

## Créer un modèle

1. Renseignez le **nom** du modèle.
2. Définissez les **horaires standard** (début et fin).
3. Indiquez le **nombre de bénévoles souhaité**.
4. Cochez éventuellement **« Définir comme template par défaut »** (voir ci-dessous).
5. Pour proposer une arrivée anticipée, cochez **« Activer un créneau anticipé »** et
   renseignez :
   - l'**heure d'arrivée anticipée** (avant l'heure de début standard) ;
   - une **explication** visible par les amapiens ;
   - le **nombre maximum de volontaires** pour ce créneau.
6. Touchez **[CRÉER TEMPLATE]**.

```
┌──────────────────────────────────────────────┐
│ ➕ Nouveau template                             │
│   Nom : [Livraison avec réception anticipée]   │
│   Horaires : Début [18:00]  Fin [20:00]        │
│   Bénévoles souhaités : [6]                    │
│   [ ] Définir comme template par défaut         │
│   ┄ Créneau anticipé ┄                          │
│   [✓] Activer un créneau anticipé               │
│   Heure d'arrivée anticipée : [17:00]          │
│   Explication : [Réception des légumes…]        │
│   Max volontaires : [2]                         │
│   [CRÉER TEMPLATE]                             │
└──────────────────────────────────────────────┘
```

## Le modèle par défaut

Un seul modèle peut être marqué **par défaut** à un instant donné. Il est alors
**sélectionné automatiquement** lorsqu'un coordinateur crée une nouvelle livraison.
Ce réglage ne modifie pas les livraisons déjà créées.

## Modifier ou supprimer un modèle

Sur un modèle existant :

- **[MODIFIER]** — ajuster ses paramètres. Les modifications **ne s'appliquent pas
  rétroactivement** aux livraisons déjà créées avec ce modèle.
- **[SUPPRIMER]** — possible uniquement si aucune livraison future n'y est associée.
- **[VOIR LIVRAISONS]** — lister les livraisons qui utilisent ce modèle.

## Comment les coordinateurs s'en servent

Lors de la création d'une livraison, le coordinateur sélectionne un modèle, qui
pré-remplit les horaires, le minimum de bénévoles et l'éventuel créneau anticipé. Il
peut ensuite **surcharger** ces valeurs pour une livraison précise, sans modifier le
modèle. Voir [Planifier une livraison](../coordinateur/01-planifier-une-livraison.md).
