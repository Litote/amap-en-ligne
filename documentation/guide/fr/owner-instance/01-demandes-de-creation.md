# Valider les demandes de création

## À quoi ça sert

Examiner les demandes de création d'une nouvelle AMAP ou d'un compte producteur, puis
les **approuver** ou les **refuser**. L'approbation déclenche l'activation du compte
administrateur de l'organisation.

## Y accéder

Depuis le tableau de bord, touchez **[VOIR LES DEMANDES]** (ou l'entrée
**« Demandes d'organisation »**).

## Les deux onglets

Les demandes sont réparties en deux onglets : **[AMAP]** (actif par défaut) et
**[PRODUCTEURS]**. Un filtre par statut permet d'afficher les demandes en attente,
approuvées ou refusées.

```
┌──────────────────────────────────────────────┐
│  [ AMAP ]  [ PRODUCTEURS ]                     │
│  Filtrer par statut : [EN ATTENTE ▼]           │
│  En attente  AMAP des Pins   Jean Dupont       │
│              jean@exemple.fr   [EXAMINER →]     │
│  Approuvée   AMAP du Lac      Paul Bernard      │
└──────────────────────────────────────────────┘
```

## Examiner une demande

1. Touchez **[EXAMINER →]** sur la demande.
2. Consultez les informations : organisation (nom, type, fuseau, langue), compte
   administrateur demandé (prénom, nom, e-mail), date et référence de la demande.

## Approuver

Touchez **[APPROUVER]**. La demande est marquée comme approuvée et le compte
administrateur de l'organisation est activé (la personne reçoit un e-mail pour définir
son mot de passe).

## Refuser

1. Touchez **[REFUSER]**.
2. Saisissez éventuellement un **commentaire** (il sera transmis au demandeur par
   e-mail).
3. Touchez **[CONFIRMER LE REFUS]**.

## Bon à savoir

- Une demande ne peut être traitée (approuvée ou refusée) **qu'une seule fois**.
- Si une demande a déjà été traitée par un autre administrateur entre-temps, le bouton
  d'approbation est désactivé et un message vous l'indique.
