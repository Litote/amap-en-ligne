# Inviter un autre administrateur

## À quoi ça sert

Inviter une nouvelle personne à devenir **administrateur de l'instance**. L'invitation
envoie un e-mail d'activation ; à l'activation, le compte obtient le rôle
d'administrateur d'instance.

## Y accéder

Depuis le tableau de bord, touchez **« Nouvel Administrateur »**.

## Envoyer une invitation

1. Renseignez le **prénom**, le **nom** et l'**e-mail** de la personne.
2. Lisez le rappel des conséquences de l'invitation (voir ci-dessous).
3. Touchez **[ENVOYER L'INVITATION]**.

```
┌──────────────────────────────────────────────┐
│  Inviter un nouvel administrateur de l'instance│
│   Prénom * [……]                                 │
│   Nom *    [……]                                 │
│   Email *  [……]                                 │
│   [ANNULER]        [ENVOYER L'INVITATION]      │
└──────────────────────────────────────────────┘
```

Après l'envoi, un écran de confirmation propose **[VOIR LA LISTE DES UTILISATEURS]** ou
**[INVITER UN AUTRE OWNER]**.

## Ce qu'il faut savoir

- Un e-mail d'activation est envoyé (lien valable 7 jours). Tant que la personne n'a pas
  activé son compte, il apparaît avec le statut **« Invité »**.
- À l'activation, le rôle d'administrateur d'instance est attribué. C'est un rôle
  **exclusif** : aucune appartenance à une AMAP ni rattachement producteur n'est créé.
- Les autres administrateurs d'instance sont notifiés.
- Si l'invitation expire sans activation, le compte est purgé et vous pouvez réémettre
  une invitation pour la même adresse.

### Messages possibles

| Cas | Message |
|-----|---------|
| Adresse déjà rattachée à un compte | « Cette adresse email correspond déjà à un compte sur l'instance… » |
| Invitation déjà en attente | « Une invitation est déjà en attente pour cette adresse… » |
| Erreur réseau | « L'envoi de l'invitation a échoué. Veuillez réessayer. » |
