# Démarrer avec l'application

Cette page vous accompagne de la toute première ouverture de l'application jusqu'à
votre tableau de bord personnel.

## À quoi ça sert

Avant de pouvoir consulter votre planning ou gérer votre AMAP, vous devez :

1. choisir le **serveur** qui héberge votre AMAP ;
2. **créer** ou **activer** votre compte ;
3. vous **connecter**.

## L'application en deux mots

Amap en ligne est gratuite, libre et fonctionne sur **téléphone Android, iPhone et
dans un navigateur web**. Elle fonctionne aussi **hors connexion** : vous pouvez
consulter vos informations et préparer vos actions sans réseau, l'application se
synchronise automatiquement dès qu'une connexion est disponible (voir
[Travailler hors connexion](#travailler-hors-connexion)).

## Choisir son serveur

Chaque AMAP est hébergée sur un **serveur**. Amap en ligne propose un serveur par
défaut, mais une AMAP peut être installée sur son propre serveur.

Au premier lancement, l'application vous propose de **choisir un serveur**. Votre
choix est mémorisé : vous n'avez pas à le refaire à chaque ouverture. Si votre AMAP
utilise son propre serveur, votre coordinateur vous communiquera l'adresse à utiliser.

## L'écran d'accueil

L'écran d'accueil (avant connexion) propose trois entrées :

```
┌─────────────────────────────────────────┐
│            🥕 Amap en ligne             │
├─────────────────────────────────────────┤
│  👤 J'ai déjà un compte                  │
│     [SE CONNECTER]                       │
│                                          │
│  🔍 Je veux rejoindre une AMAP           │
│     [S'INSCRIRE À UNE AMAP]              │
│     Choisir une AMAP : [ … ▼]            │
│                                          │
│  🏢 Je veux créer une nouvelle           │
│     organisation (AMAP ou producteur)    │
│     [INSCRIVEZ-VOUS]                     │
└─────────────────────────────────────────┘
```

- **[SE CONNECTER]** — vous avez déjà un compte (voir [Se connecter](#se-connecter)).
- **[S'INSCRIRE À UNE AMAP]** — vous souhaitez rejoindre une AMAP existante
  (voir [Rejoindre une AMAP](#rejoindre-une-amap)).
- **[INSCRIVEZ-VOUS]** — vous souhaitez créer une nouvelle AMAP ou un compte producteur
  (voir [Créer une organisation](#créer-une-organisation-amap-ou-producteur)).

## Se connecter

1. Sur l'écran d'accueil, touchez **[SE CONNECTER]**.
2. Saisissez votre **e-mail** et votre **mot de passe**.
3. Vérifiez que le **serveur** indiqué est bien celui de votre AMAP.
4. Cochez éventuellement **« Se souvenir de moi »** pour rester connecté plus longtemps.
5. Touchez **[SE CONNECTER]**.

Après connexion, vous arrivez automatiquement sur le tableau de bord correspondant à
votre rôle.

> **Première connexion ?** Vous devez d'abord avoir reçu une invitation par e-mail de
> votre coordinateur, ou avoir activé votre compte (voir ci-dessous). Sans invitation,
> contactez votre AMAP.

## Activer son compte

Lorsqu'un administrateur vous invite, ou après l'approbation de votre demande de
création d'organisation, vous recevez un **e-mail d'activation**.

1. Ouvrez l'e-mail et touchez le lien d'activation.
2. Choisissez un **mot de passe** (au moins 8 caractères) et confirmez-le.
3. Touchez **[ACTIVER MON COMPTE]**.
4. Une fois le message « Compte activé » affiché, touchez **[SE CONNECTER]**.

> Le lien d'activation a une durée de validité limitée (en général 7 jours). S'il a
> expiré, demandez à votre administrateur de vous renvoyer une invitation.

## Mot de passe oublié

1. Sur l'écran de connexion, touchez **« Mot de passe oublié ? »**.
2. Saisissez votre e-mail et touchez **[ENVOYER LE CODE]**. Un code à 6 chiffres vous
   est envoyé (valable 1 heure).
3. Saisissez le **code reçu**, choisissez un **nouveau mot de passe** et confirmez-le.
4. Touchez **[RÉINITIALISER]**.

Une fois le mot de passe réinitialisé, vous revenez à l'écran de connexion.

## Rejoindre une AMAP

Vous n'avez pas encore de compte et souhaitez rejoindre une AMAP existante :

1. Sur l'écran d'accueil, choisissez votre AMAP dans la liste **« Choisir une AMAP »**.
2. Touchez **[S'INSCRIRE À UNE AMAP]** et renseignez les informations demandées.

Votre demande est transmise à l'AMAP. Après acceptation par un administrateur, vous
recevez une invitation par e-mail pour activer votre compte.

## Créer une organisation (AMAP ou producteur)

Vous souhaitez créer une nouvelle AMAP, ou un compte producteur :

1. Sur l'écran d'accueil, touchez **[INSCRIVEZ-VOUS]**.
2. Choisissez le **type d'organisation** : **AMAP** ou **Producteur**.
3. Renseignez le **nom**, le **fuseau horaire** et la **langue** de l'organisation.
4. Renseignez votre **identité** (prénom, nom, e-mail) : vous serez le premier
   administrateur de cette organisation.
5. Acceptez les conditions d'utilisation, puis touchez **[CRÉER]**.

Votre demande est examinée par l'équipe de l'instance (réponse habituelle sous
quelques jours ouvrés). En cas d'approbation, vous recevez un e-mail d'activation pour
définir votre mot de passe et accéder à votre espace.

## Naviguer dans l'application

Une fois connecté, le **menu principal** s'ouvre via le bouton **[Menu]** en haut de
l'écran. Son contenu s'adapte à vos rôles.

- Si vous cumulez plusieurs rôles (par exemple bénévole et coordinateur), le menu
  regroupe les entrées par section de rôle.
- Les entrées **[PRÉFÉRENCES]** et **[DÉCONNEXION]** sont toujours présentes en bas
  du menu.

## Régler ses préférences et notifications

Depuis le menu, **[PRÉFÉRENCES]** vous permet de :

- consulter et modifier vos **informations personnelles** (nom, e-mail, téléphone) ;
- choisir vos **rappels** avant un créneau de bénévolat (24 h, 2 h, 30 min avant) ;
- activer ou non les **alertes** (besoin urgent de bénévoles, modifications de planning) ;
- choisir vos **canaux de notification** : notifications sur le téléphone et/ou e-mail.

L'e-mail reste toujours disponible : au moins un canal doit rester actif. Le SMS n'est
pas proposé.

Touchez **[ENREGISTRER LES MODIFICATIONS]** pour valider.

## Travailler hors connexion

L'application est conçue pour fonctionner **même sans réseau** :

- vos données (planning, contrats, membres…) sont disponibles localement ;
- les actions que vous réalisez (inscription, modification…) sont **mises en file
  d'attente** et envoyées automatiquement dès qu'une connexion est rétablie ;
- la synchronisation se déclenche seule (au démarrage, après une action, à la
  reconnexion) ; un bouton de **synchronisation manuelle** est aussi disponible sur
  certains écrans.

Quand le serveur est injoignable (pas de réseau, coupure…), l'application vous le
signale par un message : « Serveur injoignable. Vos modifications sont enregistrées
sur cet appareil et seront synchronisées dès que la connexion sera rétablie, tant que
vous ne vous déconnectez pas. » Vos modifications restent bien enregistrées
localement — attention toutefois : **se déconnecter efface les modifications en
attente d'envoi**.

Si une action est refusée par le serveur (par exemple parce qu'une information a changé
entre-temps), l'application vous l'indique pour que vous puissiez recommencer.
