# Documentation

## Architecture

Architecture guidelines are located in the [architecture](architecture/README.md) directory.

## Features

Features are described in French in the [features](feature/fr/README.md) directory.

This includes the UI specifications for coordinator season contract definition, member contract consultation, and coordinator contract assignment in `feature/fr/ui/`.

## User guide

End-user documentation, written in French and organised by role (member, coordinator,
producer, AMAP admin, instance admin), lives in the [user guide](guide/fr/README.md)
directory. Unlike the feature specs, the guide targets the app's final users and uses
no English technical terms.
